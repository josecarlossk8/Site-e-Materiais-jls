# Guia de Uso no Codex

## Ideia principal

Use esta pasta como memoria central da JLS. O Codex pode criar site, copy, roteiro, proposta, PDF, campanha e abordagem comercial, mas a qualidade melhora quando toda nova tarefa parte do mesmo contexto.

## Quando continuar nesta conversa

Continue a mesma conversa quando o assunto for desdobramento direto do que esta sendo feito agora.

Exemplos:

- Ajustar copy de uma secao do site.
- Refinar um PDF ja criado.
- Corrigir um CTA.
- Melhorar uma abordagem comercial existente.
- Revisar uma lista de contas que ja esta em andamento.

## Quando abrir uma nova conversa

Abra uma nova conversa quando o trabalho for uma frente com vida propria.

Exemplos:

- Planejamento mensal de conteudo.
- Criacao de campanha para e-commerce.
- Playbook comercial para prospeccao ativa.
- Desenvolvimento de uma nova pagina do site.
- Criacao de proposta comercial padrao.
- Estruturacao de funil, CRM ou rotina de follow-up.

## Como iniciar uma nova conversa

Use um pedido assim:

> Leia `central_jls/00_contexto_master/contexto_master.md` e use a Central JLS como base. Quero trabalhar em [tema] para [publico/nicho], com objetivo de [resultado esperado].

## Frentes recomendadas

| Frente | Quando usar | Arquivos base |
|---|---|---|
| Site e deploy | Pagina, dominio, Vercel, links, formularios | `01_site_e_materiais` |
| Conteudo | Posts, artigos, calendario, campanhas | `03_conteudo_marketing` |
| Comercial | Abordagens, objeções, proposta, scripts | `02_estrategia_comercial` |
| Prospecção | Cadencias, listas, segmentos, contas-alvo | `04_prospeccao` |
| Operação | Servicos, SLA, cotacao, briefing de carga | `05_operacao_e_produtos` |
| Provas e cases | Chancelas, historias, evidencias | `06_cases_e_provas` |
| Google Perfil da Empresa | Perfil JLS, Azul Logistica, fotos, avaliacoes | `07_google_perfil_empresa` |
| Campanhas e rotina comercial | Campanhas, funil, CRM simples, cadencias | `08_campanhas_rotina_comercial` |

## Regra de versao

Quando uma ideia ficar boa, salve como arquivo na central. Conversa solta ajuda no momento, mas arquivo salvo vira patrimonio da JLS.

## Regra de decisao

Se uma sugestao parecer bonita mas nao ajudar um decisor B2B a entender, confiar ou chamar a JLS, ela deve ser simplificada.
